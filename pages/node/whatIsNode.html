Node.js簡介 Node是什麼 ？伺服器？ Node.js is a JavaScript runtime built on Chrome’s V8 JavaScript engine. Node.js不是一門語言 Node.js不是函示庫，不是框架 Node.js是一個JavaScript運行時環境 簡單說用以解析與執行JavaScript代碼 （類似瀏覽器）-脫離瀏覽器 以前只有瀏覽器可解析執行js 瀏覽器的JavaScript EcmaScript(基本語法if,var,function……) Bom Dom Node.js中的JavaScript 沒有Dom,Bom（不處理前端頁面） 有EcmaScript 在Node.js執行環境中提供Server級別的操作api(如文件讀寫，網路服務構建…) Node.js使用event-driven,non-blocking I/O event-driven事件驅動 non-blocking I/O非阻塞IO(異步操作api) 達到輕量，高效服務 npm套件管理工具 Node.js生態系以npm開源提供使用 絕大多數第三方套件透過npm存取-方便使用者存取
